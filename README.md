# OTSGE Tap

## How do I install these formulae?

`brew install --formula otsge/tap/<formula>`

Or `brew tap otsge/tap` and then `brew install --formula <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "otsge/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
